# Kokoro TTS Models

This folder should contain the following model files:

- `kokoro-v1.0.onnx` (~330MB)
- `voices-v1.0.bin` (~330MB)

## Download Instructions

1. Visit: https://huggingface.co/hexgrad/Kokoro-82M/tree/main
2. Click on "Files and versions" tab
3. Download both files and place them here

These files are too large for Git and are excluded via .gitignore.
